**Credit Risk Scorecard Development (Lending Club)
Project Overview**

Developed an end-to-end credit risk model to predict loan default probability using Lending Club dataset.

**Key Steps**

Data cleaning & preprocessing

Feature engineering (credit burden, installment-income ratio)

WOE transformation & IV analysis

Logistic Regression modeling

**Model evaluation using:**

AUC

KS Statistic

Gini

Decile analysis

**Model Performance**

Test AUC: ~0.69

KS Statistic: ~0.28

Gini: ~0.39

**Techniques Used**

Python

Pandas

Scikit-learn

Weight of Evidence (WOE)

Information Value (IV)

Credit Risk Modeling

**Business Objective**

Build an interpretable credit risk scorecard aligned with industry practices.
